# Personal Website (Astro)

## Local

```bash
npm install
npm run dev
```

## Netlify UI settings

- Branch to deploy: `main`
- Base directory: `personal-website`
- Build command: `npm run build`
- Publish directory: `personal-website/dist`
- Functions directory: leave default (`netlify/functions`) unless you use Netlify Functions

No environment variables are required for this static site.
